# EncryptionTool

C#で作成された多機能暗号化・復号化ツール (WPFアプリケーション)

## 機能

このツールは以下の暗号化アルゴリズムとハッシュ関数をサポートしています：

### 暗号化アルゴリズム
- **AES暗号化** - 推奨される最新の対称鍵暗号化方式
- **DES暗号化** - レガシーサポート用（⚠️ セキュリティが低く、重要なデータには非推奨）
- **TripleDES暗号化** - レガシーサポート用（⚠️ 新規開発ではAESを推奨）
- **Base64エンコード** - テキストのエンコード/デコード

### ハッシュ関数
- **MD5ハッシュ** - チェックサム用（⚠️ セキュリティ用途には非推奨）
- **SHA256ハッシュ** - セキュアなハッシュ関数
- **SHA512ハッシュ** - より強固なセキュアハッシュ関数

## 使用方法

1. アプリケーションを起動
2. ドロップダウンメニューから使用したいアルゴリズムを選択
3. 暗号化セクションにテキストを入力
4. 「暗号化」または「ハッシュ化」ボタンをクリック
5. 結果をコピーまたは復号化セクションで復号化

## セキュリティに関する注意事項

⚠️ **重要**: このツールはデモンストレーションと学習目的で作成されています。

- 暗号化キーがソースコードにハードコードされています
- 本番環境では、適切な鍵管理システムを使用してください
- DESとMD5は暗号学的に安全ではないため、セキュリティが重要な用途には使用しないでください
- AES暗号化とSHA256/SHA512ハッシュを推奨します
